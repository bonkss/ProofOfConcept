Keyboard Documentation: How to use the Site with a Keyboard

How can I Interact with the Site using a Keyboard?

On the Homepage, You can press the down key to navigate to the only interactavle button on the Page. This is the "Sort Kitties" Button.

Upon Press, the button will be highlighted and clicking the enter key will allow the User to interact as a click, with only using a keyboard.



Things to Read, You can press the down arrow and the up arrow to navigate accross the list of links.

Either of these button presses with start you at the begining of the list.

Selecting the up arrow allows the user to navigate upwards in the list of links. While the down arrow navigates the user down the list.

By pressing the Enter key, the user can select the link that is highligted with their key inputs to navigate to the linked page



Types of Usability Principles

Feedback: When the user make a keyboard input (the arrows keys), elements are highlighting, showcasing that their action has been acomplished

Constraint: Users are keyboard interactivity can only be done in areas that are interactable. The user's interface is simplified and know exactly which areas are interactable and where the next actions are appropriate.

Consistency: Keyboard interactviity are the same in each page. Arrows keys to move around interactable elements and enter to click on the element. Users will only need to learn how the keyboard interaction work once
